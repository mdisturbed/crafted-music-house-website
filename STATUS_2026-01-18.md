### 1. Schema.org Structured Data (+6 SEO points)
**Why**: Helps Google understand your content as music/artists, not just text
**Impact**:
- Rich search results (star ratings, links, artist info)
- Google Knowledge Panel eligibility
- Better voice assistant integration
- 30-40% higher click-through rates

**What to Add**:
```json
// In index.html - Organization/MusicGroup schema
{
  "@type": "MusicGroup",
  "name": "Crafted Music House",
  "genre": ["Jazz", "Blues", "Americana"],
  "url": "https://craftedmusichouse.com"
}

// In ArtistProfile.tsx - Person schema (per artist)
{
  "@type": "Person",
  "name": "Louis 'Sunny' Boudreaux",
  "genre": "Jazz",
  "memberOf": "Crafted Music House"
}

// In Discography.tsx - Album/Track schema
{
  "@type": "MusicAlbum",
  "name": "After the Applause",
  "byArtist": "Salvatore Moretti"
}
```

**Estimated Time**: 2 hours
**Files to Modify**: `index.html`, `ArtistProfile.tsx`, `Discography.tsx`
